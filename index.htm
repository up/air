air - adaptive image resizing

air is a tiny tool (specifically for responsive web design) that can significantly improve the performance of the mobile version(s) of your website. Resize this page to see it in action!

How does it work?

In short: air loads different images for different window sizes. You can decide how many alternative pictures you will offer and which breakpoints you will define.

At a glance

  • Tested with Chrome 17, Firefox 11, Safari 5, IE 7+8

  • Lightweight (1.36KB / 824 bytes gzipped)
  • Easy to integrate and use
  • Compatible with the following approaches: Resposive (Web) Design, Mobile First, Progressive Enhancement

Example 1

<script src="air.js"></script> <script>
air.set({
  1: '-base',
  321: '-small',
  641: '-medium',
  961: '',
  1280: '-wide'
});
</script>

Explanation

  • Suffix '-base' between 1 and 320 px display width
  • Suffix '-small' between 320 and 640 px display width
  • Suffix '-medium' between 640 and 960 px display width
  • No suffix between 960 and 1280 px display width
  • Suffix '-wide' for display width greater 1280 px
